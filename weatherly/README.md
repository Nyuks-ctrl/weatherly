# Weatherly

A modern, simple, vibe coded weather application built with React and Capacitor.

## Features
- Auto location detection
- City search
- Animated weather icons
- Hourly forecast (iOS-style scroll)
- 7-day forecast
- Dark, modern UI
- Android & iOS support

## Tech Stack
- React + Vite
- Open-Meteo API
- Framer Motion
- Capacitor

## Platforms
- Web
- Android
- iOS

##How to run:

**1. Clone the Repo:**
   git clone https://github.com/Nyuks-ctrl/weatherly.git
   cd weatherly

   
**2.Install dependencies:**
  npm Install

  
**3.Start the development server:**
  npm run dev

  
**4. Open in browser:**
   http://localhost:5173
